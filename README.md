# Voyara - Travel Booking System 🌍

Voyara is a web-based travel booking system that provides users with a complete travel planning and booking experience. Users can explore destinations, select packages, choose hotels, select transportation modes, and complete their bookings.

## ✨ Features

- 🌎 Explore different destinations
- 📦 Travel packages selection
- 📝 Personal information booking form
- 🏨 Hotel selection with images and pricing
- 🚆 Travel mode selection:
  - Train
  - Bus
  - Airplane
  - Ship
- ⏰ Multiple departure timings
- 📍 Expected arrival time display
- 💳 Payment process
- 🎉 Booking confirmation
- 📜 Booking history

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage
